## 前言  

manager-system 项目致力于打造一个完整的后台管理系统，采用现阶段流行技术实现。  


## 项目介绍  

manager-system 是一个商城后台管理系统的前端项目，基于Vue+Element实现，采用响应式布局方式。主要包含后台面板、商品管理、用户管理、订单管理、管理员管理、系统设置、分销模块、统计报表等功能。
